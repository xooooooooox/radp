## Quick Start

本项目默认使用 Maven 来构建, 最快的使用方式是 `git clone` 到本地, 然后执行以下命令:

```shell
./mvnw clean install -T 4C -DskipTests
```

_说明: 建议使用工程内置的 `maven wrapper` 即 `mvnw` 或 `mvnw.cmd` 来进行项目的构建._
